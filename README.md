# Overview

# Build

```bash
# Create package.json if it's missing
$ npm install/i

# Make changes

# Commit changes
$ git add .
$ git commit -m 'commit message'

# Incriment version
$ npm version (major|minor|patch)

# Push
$ git push

# GitHub action (see .github) will trigger.
```

```bash
# Execute
$ node main.js

# Open up `http://localhost:3000/` in a browser
```
